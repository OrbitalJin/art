# Art Overview

Art is a desktop application combining chat and note-taking capabilities with cross-referencing between them.

## Features

### Chat System
- Multi-session support with persistent history
- Streaming responses with real-time interaction
- Model selection (Genesis, Bloom, Eden)
- Session pinning and organization
- Auto-generated session titles
- Export/import functionality

### Notes System
- Rich markdown editor with live preview
- Workspace organization (Work, Personal, Research)
- Tag system with @mention highlighting and clickable navigation
- Image support with resizing capabilities
- Auto-save with character/word count tracking
- Read/edit mode toggle

### Integration
- Cross-referencing between chats and notes
- Reference picker to add notes as context to conversations
- Unified command palette for navigation
- Import/export for both sessions and notes

## Shortcuts

### Global
| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open command palette |
| `Cmd/Ctrl + T` | Toggle theme |
| `Cmd/Ctrl + Alt + S` | Create quick session |
| `Cmd/Ctrl + Alt + N` | Create quick note |
| `Alt + 1` | Navigate to chat |
| `Alt + 2` | Navigate to notes |

### Chat
| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift + Enter` | New line in prompt |

### Notes
| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + O` | Open notes command palette |
| `Cmd/Ctrl + B` | Bold text |
| `Cmd/Ctrl + I` | Italic text |
| `Cmd/Ctrl + K` | Insert link |

## Usage

- Use `@tags` in notes for clickable navigation
- Pin important sessions for quick access
- Reference notes in chat for contextual conversations
- Use workspaces to separate different content areas
- Command palette provides universal access to features
